# PackTaskProject

**User Profile Editing Task**<br><br>
This small web application is madeup of two web pages, which shows the user details and also allows the user to edit the details along with the updating the avatar.<br><br>
# To do this project the following technologies are used.<br>
**FrontEnd**: HTML, CSS, JS<br>
**BackEnd**: Python Flask<br><br>
Additionally, the application will run on port 5040, 'http://127.0.0.1:5040/'<br><br>
# Basic Requirements to run the project: <br>
**Python** : Latest version preferred 3.11<br>
**Flask**  : Latest version preferred
